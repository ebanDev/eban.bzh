KeepassDX

KeepassDX est un gestionnaire de mots de passe développée par Kunzisoft. L'application est disponible sur Android (F-Droid & Playstore).

Exodus Privacy
3

D'après Exodus Privacy, l'application KeepassDX ne contient aucun tracker.

Politique de confidentialité
3

Vos données sensibles ne sont pas stockées sur un serveur privé ou dans un cloud fermé. Vous avez le contrôle de vos mots de passe.

Chiffrement utilisé
2

KeepassDX utilise les algorithmes AES, Twofish, ChaCha20 et Argon2. Certains de ces algorithmes ne sont pas open-source ce qui empeche de mettre à l'épreuve leurs solidité. L'application étant open source vous pouvez aller voir comment ces algorithmes sont implémentés.

Open-Source
3

L'application KeepassDX est entièrement open source. Le code est disponible sur github en cliquant sur ce lien.

Réputation de l'entreprise
3

L'application est développée par Kunzisoft avec l'aide de la communauté.
